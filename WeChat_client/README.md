## 基于深度学习算法的电力负载分类与预测系统微信小程序前端后台模块

## _由于该模块涉及企业机密，无法提供源代码，故不做开源，敬请谅解。_

### 模块功能概述
1.登录注册模块:登录:用户使用用户名和密码登录。注册:用户需提供手机号和两次相同密码。切换用户:用户可退出登录。
2.电器实时功率图表和预测功率图表的显示
